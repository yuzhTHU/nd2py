  • show_progress – 是否显示进度条

离线阶段：基于 Kruskal 最小生成森林算法的变体发现化简规则并构建 reduce_rules 字典。

这是多进程版本，使用 joblib.Parallel 加速同一长度内的表达式处理。

Parameters:

  • l_max – 最大表达式长度

  • n_jobs – 并行进程数，-1 表示使用所有 CPU 核心

  • save_cache – 是否将构建结果缓存到文件

  • show_progress – 是否显示进度条
